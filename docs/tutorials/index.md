# Tutorials

Step-by-step tutorials to help you learn Templify from basics to advanced features.

## Getting Started

### [Tutorial 1: Hello World](01-hello-world.md)
**Duration:** 30 minutes | **Level:** Beginner

Learn the basics of Templify by creating your first template:
- Setting up your project
- Creating templates with placeholders
- Processing templates with data
- Working with nested objects
- Formatting data
- Error handling

**What you'll learn:**
✓ Basic placeholder syntax
✓ Nested data access
✓ Different data types
✓ Error handling patterns

---

### [Tutorial 2: Invoice Generator](02-invoice-generator.md)
**Duration:** 45 minutes | **Level:** Intermediate

Build a real-world invoice generator with advanced features:
- Table loops for line items
- Calculations and formatting
- Multiple data sources
- Professional layouts

**What you'll learn:**
✓ Table loops with `{{#foreach}}`
✓ Complex data structures
✓ Real-world patterns
✓ Production-ready code

---
